# 🕵️ The Daily Sleuth — Wanted: Memory Match

## 📰 Project Overview

**Wanted: Memory Match** is a vintage newspaper-themed concentration game developed as the WST Final Project for **Web Systems and Technologies** at **UPHSD Molino — College of Computer Studies**.

Players flip cards to match identical pairs of suspects and clues, testing their memory and concentration. The game tracks moves and time, rewarding high-performance detectives with prestigious ranks.

## 🎮 Game Features

| Feature | Description |
|---------|-------------|
| **8 Pairs** | Match 16 cards (4x4 grid) with 8 unique evidence pairs |
| **Move Counter** | Tracks every 2-card flip attempt |
| **Timer** | Real-time elapsed display |
| **Hint System** | Temporarily reveals all unmatched cards |
| **Rank System** | S★★★ (≤14 moves) · A★★☆ (15-18) · B★☆☆ (19-24) · C☆☆☆ (≥25) |
| **Best Score** | Saves your best performance using localStorage |
| **Responsive** | Fully functional on mobile, tablet, and desktop |

## 🕹️ How to Play

1. **Start Game** — Click START or RESTART to begin
2. **Flip Cards** — Tap/click any card to reveal evidence
3. **Match Pairs** — Find two identical cards to secure a match
4. **Use Hint** — Reveals all cards for 1 second (use when stuck!)
5. **Complete Case** — Find all 8 pairs to solve the mystery
6. **View Rank** — Your detective rank appears in the victory modal
